## 🚀 طريقة التثبيت

### على Termux (موبايل):

```bash
# 1. ثبت Node.js
pkg update && pkg install nodejs git -y

# 2. انسخ البوت الأصلي
git clone https://github.com/deveni0/Pomni-AI barq-bot
cd barq-bot

# 3. احذف ملفات القديمة واستبدلها بالجديدة
# (انسخ الملفات من barq-bot.zip على مجلد barq-bot)

# 4. ثبت الباكدجز
npm install

# 5. شغّل البوت
npm start
```

## 🎮 كل الأوامر

### 🏁 الألعاب
| الأمر | الوصف |
|-------|-------|
| `.سباق` | سباق كتابة أسماء أنيمي - مين يكتب أكتر في 30 ثانية 🏁 |
| `.كويز` | سؤال أنيمي - 30 ثانية للإجابة 🎮 |
| `.خمن` | خمن شخصية أنيمي من 3 هنتات (60 ثانية) 🌸 |
| `.معركة` | شخصيتين يتقاتلوا - الجروب يصوت (30 ثانية) ⚔️ |
| `.ترتيب` | ترتيب اللاعبين بالنقاط 🏆 |

**أوامر أثناء اللعب:**
- `.هنت` ← هنت في الكويز
- `.هنت2` / `.هنت3` ← هنتات إضافية في خمن
- `.توقف` ← إيقاف أي لعبة
- `1` / `2` ← تصويت في المعركة

### 🎌 الأنيمي
| الأمر | الوصف |
|-------|-------|
| `.شخصية [اسم]` | معلومات شخصية أنيمي |
| `.اقتباس` | اقتباس عشوائي من أنيمي |
| `.توب` | أفضل 10 أنيمي |

**ردود تلقائية على:** ناروتو، ون بيس، JJK، AoT، Demon Slayer، وأكتر!

### 📥 التحميل
| الأمر | الوصف |
|-------|-------|
| `.يوتيوب [لينك أو بحث]` | تحميل فيديو يوتيوب |
| `.تيكتوك [لينك]` | تحميل فيديو تيك توك |

### 🌸 الجروب
| الأمر | الوصف |
|-------|-------|
| `.قوانين` | قوانين جروب VALK |
| `.ترحيب` | رسالة ترحيب |

### 👑 الأدمن
| الأمر | الوصف |
|-------|-------|
| `.كيك @شخص` | طرد شخص |
| `.ترقية @شخص` | ترقية لأدمن |
| `.ازالة ادمن @شخص` | إزالة صلاحية |
| `.قفل` / `.فتح` | قفل أو فتح الجروب |
| `.اسم [اسم جديد]` | تغيير اسم الجروب |

### ⚡ سونغ فقط
| الأمر | الوصف |
|-------|-------|

| `.تشغيل` / `.إيقاف` | تشغيل/إيقاف البوت في الجروب |
| `.ريستارت` | إعادة تشغيل البوت |
